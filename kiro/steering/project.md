# Project: My Dashboard (CodingCamp – Jonathan Darmawan)

## Overview
A personal browser dashboard built with vanilla HTML, CSS, and JavaScript.
No frameworks, no backend, no build tools required.

## Tech Stack
- **HTML** – structure (`index.html`)
- **CSS** – styling (`css/style.css`) — only 1 CSS file allowed
- **JavaScript** – logic (`js/app.js`) — only 1 JS file allowed
- **Storage** – browser `localStorage` only (no server)

## Folder Structure
```
dashboard/
  index.html
  css/
    style.css
  js/
    app.js
  .kiro/
    steering/
      project.md
      features.md
```

## Constraints
- No frameworks (React, Vue, etc.)
- No backend or server
- No test setup required
- Must work in Chrome, Firefox, Edge, Safari
- Keep code clean and readable

## Deployment
- Source hosted on GitHub
- Published via GitHub Pages
